# Cheat Sheet

## Claude Code (Terminal)
- Open terminal: `claude`
- Clear conversation: `/clear`
- Resume last conversation: `claude --continue`

## Obsidian Quick Actions
- New note: `Cmd+N`
- Quick switcher: `Cmd+O`
- Search: `Cmd+Shift+F`
- Command palette: `Cmd+P`
- Toggle left sidebar: `Cmd+\`

## Common Queries for AI
- Active engagements: `status:active type:engagement`
- Pipeline proposals: `status:draft type:proposal`
- Client contacts: `type:person tags:client`
